# Классификация и сортировка электронной почты

Нейронная сеть для автоматической сортировки писем на категории "Работа" и "Личное".

## 📁 Файлы проекта
├── Nazarov.csv # Исходные данные
├── NS_email_classifier.ipynb # Обучение и тестирование модели
├── email_classifier.keras # Сохраненная модель (создается после обучения)
├── scaler.pkl # Сохраненный scaler (создается после обучения)
└── README.md # Описание проекта


## 📊 Данные

Файл `Nazarov.csv` содержит матрицу частот слов (Bag of Words):

- **Строки**: 3168 писем
- **Столбцы**: 3000+ признаков (частоты слов)
- **Prediction**: целевая переменная
  - `0` — Личное письмо
  - `1` — Рабочее письмо

## 🧠 Архитектура нейронной сети

| Слой | Параметры |
|------|-----------|
| Dense | 128 нейронов, ReLU |
| Dropout | 0.3 |
| Dense | 64 нейрона, ReLU |
| Dropout | 0.3 |
| Dense | 32 нейрона, ReLU |
| Dense | 1 нейрон, Sigmoid |

**Параметры обучения:**
- Оптимизатор: Adam
- Функция потерь: binary_crossentropy
- EarlyStopping: patience=10
